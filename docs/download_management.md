# 下载管理

## 功能入口

- 下载器配置: `/download/downloader`
- 下载设置: `/download/settings`
- 正在下载: `/download/downloading`
- 下载历史: `/download/history`
- 自动删种: `/download/torrent-remove`

## 下载器配置

添加和管理下载客户端：

### 支持的类型

| 类型 | 版本要求 | 说明 |
|------|----------|------|
| qBittorrent | >= 4.3.9 | 推荐，功能最全 |
| Transmission | >= 3.0 | 轻量稳定 |
| 迅雷 | - | 辅助下载，不支持刷流 |

### 关键配置项

- **监控**: 下载完成后自动转移文件到媒体库
- **标签隔离**: 仅处理含 `NEXUS_MEDIA` 标签的任务
- **目录隔离**: 仅处理指定目录范围内的任务
- **种子管理模式**: 默认/手动/自动（QB 专用）
- **下载目录**: 按媒体类型（电影/电视剧/动漫）分别配置保存路径

### 默认下载器

设置一个默认下载器，未指定时使用。点击五角星图标设为默认。

## 下载设置

预设下载参数模板，适用于常规下载任务（刷流除外）：

| 参数 | 说明 |
|------|------|
| 分类 | QB 分类标签 |
| 标签 | 自动添加的标签 |
| 上传限速 | KB/s |
| 下载限速 | KB/s |
| 分享率限制 | 达到后自动停止做种 |
| 做种时间限制 | 小时 |
| 下载器 | 指定使用的下载器 |

所有任务自动添加 `NEXUS_MEDIA` 标签。

## 正在下载

查看下载器中的活动任务：

- 实时进度和速度
- 支持暂停/开始/删除操作
- 查看种子详情

## 下载历史

查看已完成的下载记录：

- 按时间倒序排列
- 显示识别结果和转移状态
- 失败的记录可重新处理

## 自动删种

配置规则自动删除下载器中的种子：

### 删种条件

| 条件 | 说明 |
|------|------|
| 做种时间 | 超过设定小时数 |
| 分享率 | 超过设定值 |
| 上传量 | 超过设定 GB |
| 平均上传速度 | 低于设定 KB/s |
| 磁盘空间 | 剩余低于设定 GB |
| Free 到期 | 促销结束后删除 |

### 规则模式

- **或模式**: 满足任一条件即删除
- **与模式**: 满足全部条件才删除

### 注意事项

- 自动删种仅删除任务和种子文件，已转移到媒体库的文件不受影响
- 建议在刷流规则中使用，常规下载慎用
