# 新增下载器

下载器配置页面位于 [/download/downloader](/download/downloader)。

- 支持同一个下载客户端添加为多个下载器，并在不同的场景选择使用。

### QB种子管理模式

🔘 **默认模式**
- 使用Qbittorrent客户端原有设置
- Nexus Media不做任何修改

🔘 **手动模式**
- 强制开启手动管理模式
- 下载目录由Nexus Media直接指定

🔘 **自动模式**
- 强制开启自动管理模式
- 下载目录由分类标签决定
- 无标签时使用默认保存路径

⚠️ **注意事项**
- 自动模式下Nexus Media会创建对应分类
- 已有分类的保存路径将被覆盖

### 监控设置
- 下载完成后自动转移文件到媒体库
- 与[目录同步](/system/sync)功能二选一

### 转移方式
参考[转移方式说明](directory_sync.md#转移方式详解)

### 标签隔离
✅ 启用：仅处理含NEXUS_MEDIA标签的任务
❌ 关闭：处理所有下载任务

### 目录隔离
✅ 启用：仅处理指定目录范围内的任务
❌ 关闭：处理所有目录的任务

> **下载器版本要求**：QB版本要求>=4.3.9，TR版本要求>=3.0，其它版本可能存在适配问题。

## 下载设置管理

下载设置页面位于 [/download/settings](/download/settings)。

⚙️ **功能概述**：
- 预设下载参数模板
- 支持限速、标签管理
- 适用于常规下载任务（刷流任务除外）
- 自动删种规则需在[/download/torrent-remove](/download/torrent-remove)中独立配置

⭐ **默认设置**：
- 系统预设不可修改
- 点击五角星设为默认
- 未指定时自动使用默认设置

🏷️ **默认标签**：
- 所有任务自动添加`NEXUS_MEDIA`标签

## 自动删种

自动删种规则页面位于 [/download/torrent-remove](/download/torrent-remove)。

- 配置基于分享率、做种时间、上传量/下载量等条件的自动删除规则。
- 规则可关联到具体下载器，对满足条件的种子执行自动删除操作。
- 删种操作不可逆，请谨慎配置阈值条件。

## 下载器类型配置

### 迅雷

📌 **配置参数**：

| 参数 | 必填 | 说明 | 默认值 |
|------|------|------|--------|
| IP地址 | ✅ | 迅雷NAS设备的IP地址 | - |
| 端口 | ✅ | 迅雷NAS Web管理端口 | 2345 |
| 认证令牌 | ❌ | Basic认证令牌 | - |

🔑 **认证令牌获取**：
1. 登录迅雷NAS Web管理界面
2. 使用浏览器开发者工具(F12)查看网络请求
3. 在请求头中找到 `Authorization` 字段
4. 复制 `Basic base64(username:password)` 格式的值

⚠️ **功能限制**：
- 不支持速度限制设置
- 不支持标签管理
- 不支持自动删种
- 不支持种子重新校验
- 不支持获取磁盘剩余空间

✅ **支持功能**：
- 添加下载任务
- 查看下载进度
- 暂停/开始任务
- 删除任务
- 下载完成监控

> **注意**：迅雷下载器适合作为辅助下载器使用，不建议用于刷流任务。
