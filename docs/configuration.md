# 基础配置指南

## 首次配置

### 1. 管理员账号
- 默认: `admin` / `password`
- **首次登录后必须修改密码！**

### 2. TMDB API Key（必须）
- 访问 [TMDB 设置页面](https://www.themoviedb.org/settings/api) 申请 API Key
- 填入 **基础设置 → 媒体库 → TMDB API Key**
- 未配置则核心功能（识别、刮削）无法使用

### 3. 网络代理（可选）
- 格式示例:
  - HTTP: `127.0.0.1:7890`
  - SOCKS5: `socks5://127.0.0.1:8018`
  - 认证代理: `socks5://user:pass@127.0.0.1:7891`
- 默认仅用于 TMDB/Telegram 访问，站点需单独开启代理

## 媒体库配置

### 分类策略

```yaml
movie:
  华语电影:
    original_language: 'zh,cn'
  外语电影:

tv:
  国产剧:
    origin_country: 'CN,TW,HK'
  欧美剧:
    origin_country: 'US,UK'
```

### 文件处理

- **重命名格式**: 支持 `{title}`、`{year}` 等占位符
  - 示例: `电影/{title} ({year})/{title}.{ext}`
- **刮削设置**: 生成 nfo 元数据和媒体图片
  - "演职人员中文"会频繁访问豆瓣，谨慎开启

## 服务配置

### 订阅设置

| 功能 | 建议值 | 说明 |
|------|--------|------|
| RSS 周期 | >= 300 秒 | 最小间隔 300 秒 |
| 搜索周期 | >= 6 小时 | 对站点压力大 |

### 同步设置

- **媒体库同步**: 保持本地与媒体服务器状态一致
- **站点刷新时间**: 支持多种格式
  - 间隔: `12` (小时)
  - 固定时间: `08:00`
  - 时间范围: `08:00-09:00`
  - Cron: `0 */6 * * *`

## 安全设置

### 访问控制
- **源地址限制**:
  - Telegram IP: `149.154.160.0/20,91.108.4.0/22`
  - 全部开放: `0.0.0.0/0,::/0`

### API 安全
- **密钥验证**: 启用后需在 URL 添加 `?apikey=xxx`

## 高级功能

### 识别增强
- **辅助识别**: 增加识别率但耗时
- **AI 识别**: 效果佳但需付费

### 搜索优化
- **豆瓣优先**: 中文支持好但可能不一致
- **多语言搜索**: 提高资源匹配率

### 辅助功能
- **验证码识别服务器**: 用于站点签到，配置 OCR 服务地址
- **网页自动化服务器**: 用于浏览器自动化登录，配置 Chrome 服务地址
