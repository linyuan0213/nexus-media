# 行为准则

本项目致力于维护一个开放、友好、包容的社区环境。

## 我们的标准

积极参与社区的行为包括：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性的批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为包括：

- 使用带有性暗示的语言或图像，以及不受欢迎的性关注
- 挑衅、侮辱/贬损性评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 其他在专业环境中被合理认为不恰当的行为

## 执行

项目维护者有权利和责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、问题和其他贡献。维护者有权利暂时或永久禁止任何 contributor 的其他不当行为。

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公共空间代表项目或其社区时的行为。

## 联系

如有违规行为，请通过项目 issue 或邮件联系维护者。
